# Resume Builder

**Resume Builder** ‚Äî a fast, easy, and customizable web app to create professional resumes/CVs using templates and AI-assisted content suggestions.

> Build a resume in minutes, export to PDF, and customize templates to match your style.

## Files in this folder:
  - `index.html` ‚Äî demo page that loads `app.jsx` via Babel in the browser.
  - `app.jsx` ‚Äî main React component (ResumeBuilder) built to run in the browser.
  - `styles.css` ‚Äî minimal stylesheet for layout and print styles.
  - `server.js` ‚Äî tiny Node static server to serve files locally.
  - `package.json` ‚Äî contains `npm run vdev` to start the server.
  - `tempCodeRunnerFile_backup.jsx` ‚Äî backup of the original JSX source.
